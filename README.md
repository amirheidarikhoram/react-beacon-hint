## Installation

```
yarn add @floating-ui/react framer-motion react-beacon-hint
```

or

```
npm i @floating-ui/react framer-motion react-beacon-hint
```

## Inspiration

React Beacon Hint is heavily inspired by [Floating UI](https://floating-ui.com/).
